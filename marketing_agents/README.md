# Marketing Agents

AI-powered marketing automation scripts for GlobalGuard AI.

## Structure

```
01_leads/      — Find potential users on social media & forums
02_content/    — Generate social media posts, ads, articles
03_replies/    — Auto-reply to user inquiries & support
04_onboarding/ — Guide new users through first-time experience
```

## Usage

Each agent is a standalone Python script:

```bash
python marketing_agents/01_leads/find_leads.py
python marketing_agents/02_content/generate_post.py
python marketing_agents/03_replies/auto_reply.py
python marketing_agents/04_onboarding/onboarding.py
```
