# GlobalGuard AI — 跨境电商合规审计 MVP

## 产品目标
帮助中国出海卖家在 T86 政策取消背景下，快速识别产品 HS 编码、评估关税/VAT 影响，降低清关风险。

## 目标人群
- 中国跨境电商卖家（主要面向美国、欧盟市场）
- 使用小包直邮模式（De Minimis / T86 路径）的商家
- 需要在发货前快速判断合规风险的运营人员

## 三大 AI Agent 分工

### 1. 视觉归类代理 (Vision Classifier)
- 输入：产品图片（Base64 / URL）
- 能力：识别产品名称、材质、用途
- 输出：3 个候选 6 位 HS 编码及置信度
- 技术：调用 DeepSeek / GPT-4o 视觉 API

### 2. 规则检索代理 (Rule Retriever)
- 输入：HS 编码
- 能力：从本地 hs_codes.json + regulations.json 匹配对应税率、禁运限制
- 输出：该编码适用的各国关税、VAT、特殊监管要求
- 技术：本地 JSON 查询 + 缓存策略更新

### 3. 合规审计代理 (Compliance Auditor)
- 输入：产品信息 + HS 编码 + 关税规则
- 能力：综合计算预估税费，评估禁运风险，生成申报建议
- 输出：结构化合规风险报告
- 规则假设：
  - 美国：30% 从价税 或 $25/件固定税率（取高者）
  - 欧盟：基础关税 + VAT（假设 20%）
  - 禁运风险：高/中/低

## 产品路线图
- Phase 1: 基础项目 + 数据层
- Phase 2: 视觉归类 API
- Phase 3: 合规报告引擎 + 前端
- Phase 4: OpenClaw 政策监控
- Phase 5: PLG 营销钩子
